# 🎮 AluGames

Projeto desenvolvido com foco em praticar lógica de programação, manipulação do DOM e interação com elementos HTML utilizando JavaScript.

---

## 📌 Sobre o projeto

O **AluGames** é uma aplicação simples que simula o aluguel de jogos, permitindo alterar o status de disponibilidade de cada item diretamente na interface.

A ideia principal é demonstrar como o JavaScript pode controlar o comportamento da página de forma dinâmica, reagindo às ações do usuário.

---

## 🚀 Funcionalidades

- Listar jogos disponíveis
- Alterar status do jogo:
  - Disponível
  - Alugado
- Atualizar textos e estilos em tempo real
- Interação com botões
- Feedback visual para o usuário

---

## 🧠 Conceitos praticados

- Manipulação do DOM
- Eventos em JavaScript
- Condicionais (`if/else`)
- Funções
- Alteração de classes CSS dinamicamente
- Lógica de estado da aplicação

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Objetivo

Este projeto foi criado para consolidar conceitos fundamentais de JavaScript, especialmente no controle de interface e interação com o usuário.

Também serve como projeto de portfólio para demonstrar conhecimento em desenvolvimento front-end.
